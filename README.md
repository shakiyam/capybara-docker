capybara-docker
===============

[Capybara-mechanize](https://github.com/jeroenvandijk/capybara-mechanize) Docker Image

How to Use
----------

```console
docker container run --rm -t -u "$(id -u):$(id -g)" -v "$PWD":/work:ro ghcr.io/shakiyam/capybara
```
